Build ReST (.rst) files
-----------------------

* rst2pdf
	
		rst2pdf {ReST-file} -o {output-file}
		
		example:
		rst2pdf "RepetitionQuestions.rst" -o "/tmp/RepetitionQuestions.pdf"
		
		
* rst2html
	
		rst2html {ReST-file} {destination}
		
		example:
		rst2html "RepetitionQuestions.rst" "/tmp/repetitionQuestions/"
